# Twitch Flow Chat

Twitchのチャット（コメント）を動画プレイヤー上にニコニコ動画風に流すためのツールです。
Google Chrome拡張機能版とTampermonkeyスクリプト版の2種類を提供しています。

## 主な機能

- **ニコニコ風スクロール**: チャットを右から左へスムーズに流します。
- **VOD（アーカイブ）対応**: アーカイブ視聴時も、リアルタイムと同じ感覚でコメントを楽しめます。複数のスタンプ（絵文字）が含まれるメッセージも完全に対応しています。
- **柔軟なカスタマイズ**: コメントの色、フォント、不透明度、表示行数、スクロール速度などを自由に設定可能です。

## インストール方法

### 1. Chrome拡張機能版

最も簡単な方法は、[Chrome ウェブストア](https://chromewebstore.google.com/detail/twitch-flow-chat/mkffadcoaamccjdiookgcpolhclapfam) からインストールすることです。

開発版を個別にインストールする場合は以下の手順に従ってください：

1. `/twitch-flow-chat-extension` をローカルにダウンロードします。
2. Chromeの拡張機能管理画面 (`chrome://extensions/`) を開きます。
3. 「デベロッパー モード」をオンにします。
4. 「展開して読み込む」をクリックし、ダウンロードしたフォルダを選択します。

### 2. Tampermonkey版

[Tampermonkey](https://www.tampermonkey.net/) などのユーザースクリプト管理拡張機能をインストールした後、`/twitch-flow-chat_tampermonkey/twitch-flow-chat-tampermonkey.user.js` の内容を新規スクリプトとして登録してください。

## 使い方

- **表示の切り替え**: キーボードの **'C'** キーを押すことで、コメント表示の ON/OFF を即座に切り替えられます。
- **設定を開く**:
  - **Tampermonkey版**: プレイヤー右下のチャット切り替えボタンを **長押し (600ms)** するか、スクリプトマネージャーのメニューから設定を開けます。
  - **拡張機能版**: ブラウザのツールバーにあるアイコンをクリックして設定画面を開けます。

## 設定項目

| 項目                     | 説明                                                        |
| :----------------------- | :---------------------------------------------------------- |
| コメント色               | テキストの基本色を設定します。                              |
| 縁取りサイズ             | アウトラインの太さを調整します（高画質対応）。              |
| 不透明度                 | コメントの透け具合を 0.0 ～ 1.0 で調整します。              |
| 表示最大行数             | 画面上に同時に表示できるレーンの数を制限します。            |
| フォントサイズ算出モード | Auto（自動計算）または Custom（固定サイズ）を選択できます。 |
| スクロール時間           | コメントが画面を横切る時間を秒単位で設定します。            |
